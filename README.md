# EMBA Cohort 25 — By the Numbers

An animated, auto-playing HTML presentation built for the University of Maryland Robert H. Smith School of Business EMBA Cohort 25 graduation banquet (May 2026).

**Live site:** [jaclynfieldmath.ai/emba-cohort25-stats](https://jaclynfieldmath.ai/emba-cohort25-stats/)

## What It Is

A single-file animated presentation that celebrates the EMBA Cohort 25 journey through data — from who we are as a cohort to the courses, professors, and assignments that defined the 16-month program (January 2025 – May 2026).

## Slides

| # | Title | Content |
|---|-------|---------|
| 0 | **EMBA Cohort 25** | Title card |
| 1 | **Who We Are** | Cohort demographics — industry and seniority breakdowns by percentage |
| 2 | **What We Signed Up For** | Hero numbers — assignments, class hours, pages read, hotel nights, credits, days on campus, courses, professors, group projects |
| 3 | **How We Spent Our Weeknights** | Assignment highlights (cases, videos, books, presentations) + donut chart of work split by category |
| 4 | **The Courses That Kept Us Up** | Top 10 courses ranked by total Canvas assignments |
| 5 | **Our Most Well-Read Subjects** | Pages read by course (top 7) |
| 6 | **The Professors Who Shaped Us** | Lollipop chart of in-class hours per professor |
| 7 | **The MBA Vocabulary** | Word cloud generated from course readings, lecture slides, and materials |
| 8 | **Congratulations, Cohort 25!** | Closing slide |

## Features

- **Single HTML file** — no dependencies, no build step, no server required
- **Auto-playing** — advances through all slides automatically (~68 seconds total)
- **Animated charts** — counters, donut charts, bar fills, and lollipop dots all animate on slide entry
- **Keyboard controls** — `Space` to pause/resume, `←` / `→` to navigate, `R` to restart
- **Responsive** — scales to any screen size using `clamp()` throughout
- **UMD branding** — uses the official Maryland red (#E21833) and gold (#FFD200) color palette
- **Geist font** — loaded from Fontsource CDN for a clean, modern look

## Data Sources

The stats in this presentation were compiled from:

- **Canvas assignment tracker** — an Excel spreadsheet tracking all 344 assignments across 19 courses
- **Weekend calendar documents** — DOCX files for each in-person weekend session, parsed to calculate class hours and professor attribution
- **Notion class notes** — used to cross-reference professor-to-session mapping for accuracy
- **Course materials** — PDFs, PPTX slides, and DOCX readings from iCloud Drive were processed to generate the word cloud (~290 files, 5.6M characters of text)
- **Cohort directory** — titles and organizations used for the "Who We Are" demographics (shown as percentages only, no individual data)

## How It Was Built

The presentation was generated by `build_presentation.py` (not included in this repo), which:

1. Reads professor hours from a JSON data file
2. Embeds the word cloud PNG as a base64 data URI
3. Outputs a fully self-contained `index.html` with all CSS, JS, and assets inline
